# 📊 Wilcoxon Signed-Rank Test Implementation

Statistical Analysis for Experimental Comparison

## 📌 Project Description

This repository contains a detailed implementation of the Wilcoxon Signed-Rank Test, a non-parametric statistical method used to compare paired samples using Python. 

## 📦 Requirements

Install dependencies using:

```shell
pip install scipy pandas numpy tabulate math
```

## 🚀 How to Run

Open the notebook:

```shell
wilcoxon_signed_rank_test.ipynb
```

Ensure the file:

```shell
Wilcoxon-Signed-Ranks-Table.xlsx
```

is located in the correct directory (or adjust the file path inside the notebook).

Run all cells sequentially.

## ⚙️ Additional Information

In this notebook, the test is applied to two different datasets from Kaggle to demonstrate consistency and robustness of the statistical analysis.

[Diabetes Health Indicator Dataset](https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset) 

[PCOS Diagnosis Dataset](https://www.kaggle.com/datasets/samikshadalvi/pcos-diagnosis-dataset)
